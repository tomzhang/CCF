Key-Value Store API¶

This page presents the API that a CCF application must use to read and mutate the replicated key-value store.

A CCF application should define one or multiple public or private kv::Map. Then, each ccf::HandlerRegistry::Handler should use the kv::Tx transaction object to read and write to specific kv::Map.
